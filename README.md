# Pipeline Inspection Robot - STM32
This project is for an industrial robot designed to inspect pipelines using **STM32**.

### Key Features:
* **Gas Detection:** MQ-135 sensor for hazardous gas monitoring.
* **Motion Sensing:** MPU6050 for orientation and stability.
* **Motor Control:** Driving DC motors and Servos.
* **Connectivity:** HC-06 Bluetooth module for remote control.

### Hardware:
- STM32 Microcontroller
- MQ-135, MPU6050, HC-06
